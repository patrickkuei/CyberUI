<div className="responsive-container">
  <h1 className="responsive-title">Responsive Design</h1>
  <p className="responsive-subtitle">Build adaptive interfaces that work on all screen sizes</p>

  CyberUI components support **responsive design** out of the box. Instead of writing complex media queries, you can pass an object to define different values for different screen sizes.

  <hr className="section-divider" />

  ## Usage

  All CyberUI components with size props support responsive values:

  <div className="code-example">
    ```tsx
    <Button size={{ base: 'sm', md: 'md', lg: 'lg' }}>
      Responsive Button
    </Button>
    ```
  </div>

  <hr className="section-divider" />

  ## Breakpoint System

  CyberUI uses the same breakpoint system as Tailwind CSS:

  <div className="code-example">
    ```tsx
    const breakpoints = {
      base: "0px",      // Mobile first (default)
      sm: "640px",      // Small tablets
      md: "768px",      // Tablets
      lg: "1024px",     // Laptops & desktops
      xl: "1280px",     // Large desktops
      "2xl": "1536px",  // Extra large screens
    }
    ```
  </div>
